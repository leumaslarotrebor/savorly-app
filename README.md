# Savorly

Savorly is a student startup concept focused on building a food application
platform to understand product workflows and application development.

## Role
Technical Lead (Student Project)

## Responsibilities
- Planned application workflows and user journeys
- Worked on UI flow design and feature ideation
- Coordinated technical implementation at a beginner level
- Learned basic backend concepts during development

## Tech Stack
- Figma (UI/UX Design)
- Python (learning phase)
- HTML, CSS (basic understanding)

## Status
This project is in an early learning stage and was built as a student initiative.
